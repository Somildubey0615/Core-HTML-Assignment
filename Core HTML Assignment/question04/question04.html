1. img tag: img tag is used to display image on the web page.it's an empty tag that contains attributs only not closing tag. attributes of img tag are (src,alt,width,hight)

  2. a tag: a tag (anchor tag) defines a hyperlink that open page to another page.it can creat hyperlink to other webpages, files or url. ("href") is the most important attribute of a tag.

  3. p tag : p(paragraph) tag is used to define a paragraph in a webpage.it always takes new paragraph.

  4. div tag : div tag is a container that is used to group and structure for styling purpose.it defines a section or divison in an HTML document.
  5. br tag : br tag is used to insert a single line break.its an empty tag, it has no closing tag.

  6. hr tag : it is used to insert horizontal line break in html page it is also an empty tag.

  7. ol tag : ol stands for orderd list. in ol all the list items are marked with numbers by default. ("li") is its attribute.

  8. ul tag : ul stands for unorderd list. in ul all the list items are marked with bullets. ("li") is its attribute.

  9. form tag: it is the most important tag of html. it contains controles such as text field, password,radio button,submit button etc. developer can take input from user by form tag.

  10. audio and video tag : the audio and video tag is used to embed sound and video content in a document, such as music audio and video.

  11. header tag: header tag is used to define a container to set Navigation bar hyper links ,logo,site title in a web page.

  12. footer tag : footer tag is used to define the footer of a webpage .it contains the information about author, copyright or other relevent detail.
  13. style tag : style tag is used to define style for document or we can say it used to apply css rules on document.
  14. aside tag : aside tag defines some content aside (indirectly related)from content.
  15. button tag : button tag is used for define a clickable button.
